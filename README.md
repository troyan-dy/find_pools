# find pools 

https://find-pools.herokuapp.com/
## Dependencies
- docker-compose
- python 3.9

## Commands
### Install
- `make install`

### Run tests
- `make test`
### Linting
- `make lint`

### Formatting
- `make format`

### Start in docker
- `make start`
- `make stop`


## Usage

- Curl
```bash
curl -X GET http://0.0.0.0:8000/ping
```

- Response

```bash
pong
```
